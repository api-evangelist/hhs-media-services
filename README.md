# HHS Media ServicesThis is a repo for managing machine readable artifacts for the [HHS Media Services APIs](https://api.digitalmedia.hhs.gov/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [HHS Media Services APIs](https://api.digitalmedia.hhs.gov/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://api.digitalmedia.hhs.gov/)- [Developer](https://api.digitalmedia.hhs.gov/)The [APIs.json](https://github.com/api-evangelist/hhs-media-services/blob/master/apis.json) index provides links to all the relevant links for the HHS Media Services APIs, including the OpenAPI and Postman if they are present.
